# Sarofiem & Antoun Intake System

## Tech Stack
- React + Tailwind
- Node.js + Express + Prisma
- PostgreSQL, OpenAI, Socket.IO

## Setup
1. `npm install`
2. `npx prisma migrate dev`
3. `npm run dev`

## Features
- Multilingual intake
- AI document generation
- E-signature lifecycle
- Lead chase FSM
- NEOS export integration
